PART-A

Computer.HDL

--> computer's.hdl file consists of 3 memory segments they are: CPU, Memory, and Rom32k.
--> First, we must set up the nand2tetris file in our system. There is a program named Hardware Simulator in folder 'tools' in nand2tetris.
--> Then computer.hdl should be uploaded. In order to operate the computer, we should also upload the test script file that was created by the user or the built-in test script file.
--> Finally, we have to run it while maintaining the maximum run-time and we must switch from the screen to the output view.

PART-B

Implementation Of Symbol In Jack

--> The above jack implementation is done with a software called VS code.
--> The above code in jack is divided into three sections.
--> The three parts stated above are:
    -Compare_two_strings() 
    -Display_Symbol_table() 
    -Define_attribute() 
--> All three of the above functions are called in the main function section.
--> To run these two Jack files, we must compile them using command prompt and then obtain a VM file.
--> The nand2tetris programme must then be installed. 
--> In nand2tetris, there is a 'tools' folder in that there is software called VM emulator.
--> The generated .vm file must then be uploaded.
--> We must change the animate from programme flow to no animation in the VM emulator's top menu bar before running the file.
--> So, finally the output will be displayed on the screen.
